# portal-kb-forge

Локальный инструмент для полуавтоматической подготовки базы знаний к загрузке в корпоративный портал.

## Что делает

- загружает DOCX;
- режет документ по ручным маркерам;
- создает карточки знания;
- фиксирует тип объекта: статья, сценарий, справочная информация, быстрый ответ, термин;
- экспортирует DOCX и `content_map.xlsx`.
